# Experiment Logs

One file per completed experiment: `exp-XX-name.md`

## Template
```markdown
# exp-XX: <name>

## Hypothesis
What we expected this strategy to do better.

## Setup
- Game number, tick started, tick ended
- Opponent count / AI vs human

## Results
- Final star count, rank
- Key decisions that worked / didn't

## Learnings
What to change in the next experiment.
```
