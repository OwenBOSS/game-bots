# notes/

## Purpose
Research notes, API reference, and experiment logs. Read-only for the bot —
nothing here is imported by np_lib or any experiment.

## Key Files
- `api.md` — reverse-engineered API endpoints and scanning_data JSON schema
- `mechanics.md` — game mechanics reference (combat math, economy, tech priority)

## Conventions
- Experiment logs go in `experiments/exp-XX-name.md` after a run completes
- When you confirm an undocumented order endpoint via DevTools, add it to `api.md`
  under "Orders (confirmed)" and remove it from "Orders (unconfirmed)"
