# Experiments Log

Use this folder to track what you've tried, what worked, and what didn't.

## Template

Create a new file for each experiment: `YYYY-MM-DD-experiment-name.md`

```markdown
# [Experiment Name]

**Date:** YYYY-MM-DD  
**Strategy:** economy-first / rush / adaptive  
**Branch:** screeps branch name  

## Hypothesis
What do you expect to happen?

## Setup
- Room: W?N? or simulation
- Starting conditions: fresh spawn, GCL X, etc.
- Key parameters changed

## Results
- Tick count at key milestones
- RCL reached / enemy defeated / outcome
- Screenshots or console logs

## Observations
What surprised you? What worked? What didn't?

## Next Steps
What to try next based on these results.
```
